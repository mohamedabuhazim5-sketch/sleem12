# Sleem

Complete Next.js project scaffold based on the uploaded consolidated codebase.

## Run locally

1. Install dependencies:
   npm install
2. Update `.env.local` with your real Supabase and WhatsApp values.
3. Execute SQL from `database/schema_and_policies.sql` in Supabase.
4. Start development server:
   npm run dev
